# news/

Drop `.md` files here to improve any PPT skill.

Each file should contain a relevant article, case study, or data point.
Name files descriptively: `2026-03-policy-update.md`, `2026-03-industry-report.md`, etc.

To trigger learning for a specific skill:
- `/learn-policy-analysis`
- `/learn-industry-analysis`
- `/learn-self-analysis`
- `/learn-overview`
- `/learn-positioning`
- `/learn-strategy`
- `/learn-operations`
- `/learn-roadmap`
- `/learn-org-structure`
- `/learn-kpi`
- `/learn-settlement`
- `/learn-resources`
- `/learn-ops-architecture`
- `/learn-tech-architecture`
- `/learn-business-model`
- `/learn-key-scenarios`
- `/learn-future-goals`

The skill will scan all `.md` files, score relevance to its topic, and rewrite its `[Examples]` section.
